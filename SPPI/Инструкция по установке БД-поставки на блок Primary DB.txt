
1. Введение (описательная часть).
1.1. В дистрибутиве передаются скрипты создания:
	- структуры основной БД приложения ЕФС. Анализ и аналитика. SPPI Тестирование.

2. Требования.
2.1. Для выполнения скриптов небходима JRE версии не ниже 1.6.
2.2. Запуск скриптов осуществляется под пользователем UFSADMIN c помощью автоматического запуска Liquibase-скриптов.

3. Предварительные настройки. 
3.1. Для наката дистрибутива необходимо:
	- Изменить URL и пароль доступа к БД в файле liquibase.properties
	- Запустить исполняемый файл dbupdate.bat.

4. Валидация.

4.1. В результате успешной работы в логе не должно быть ошибок.
4.2. В конце лога «DBUpdate.log» должна быть строка «liquibase: Successfully released change log lock».
4.3. В таблице UFSSYS.LIQ_DATABASECHANGELOG должны появиться записи о запуске и накате изменений.
